## Emphasis

### Bold
Bold text is denoted using two `*` symbol. For example:
`**Astro**`
would be: I Love **Astro!**

### Italic

Italics are denoted with `_` symbols on either side of the word.

for example `I am a _Web Developer_.` would be: I am a _Web Developer_.

### Bold and Italic

Italic and bold symbols can be combined.

for example: `Thou **_SHALL_** not touch.` would be: Thou **_SHALL_** not touch.

## Lists

### Unordered

- Create a list by starting a line with `+`, `-`, or `*`
- Sub-lists are made by indenting 2 spaces:
  - Marker character change forces new list start:
    - Ac tristique libero volutpat at
    - Facilisis in pretium nisl aliquet
    - Nulla volutpat aliquam velit
- Very easy!

## Image

All images can be directly used in the document with the `<Image>` tag. 

all images require a source, alternate text, width, and height. <br/>
The width and height will be set at `width = 1200` and `height = 630` so try to use high resolution pictures for the best result.

an example image embed: 

`<Image src="https://uploads.dailydot.com/2018/10/olli-the-polite-cat.jpg?auto=compress&fm=pjpg" alt="memecat.png" width="1200" height="630" />`
